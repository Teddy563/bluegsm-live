Intrebari Frecvente

Cat dureaza o reparatie?

Majoritatea operatiunilor (schimb display, baterie) se realizeaza pe loc (30-60 minute) daca piesa este pe stoc. Pentru interventii complexe pe placa de baza, timpul poate varia intre 24-48h.

Oferiti garantie?

Da! Oferim garantie intre 3 si 12 luni pentru orice piesa inlocuita si pentru manopera. Bonul fiscal si certificatul de garantie se ofera pe loc.

Se pierd datele din telefon?

In 99% din cazuri (schimb display, baterie, mufa), datele tale raman intacte. Totusi, recomandam intotdeauna un backup preventiv inainte de a veni in service.
